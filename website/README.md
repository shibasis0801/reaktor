# Reaktor Website

This directory contains the Docusaurus site scaffold for Reaktor documentation.

## Current status

The canonical docs for active development are still the GitHub Markdown docs in the repo root and module folders.

This website folder is useful if you want to:
- turn the Markdown docs into a browsable docs site
- publish product/framework docs externally later

## Local development

```bash
yarn
yarn start
```

## Build

```bash
yarn build
```

## Note

Some of the Docusaurus tutorial pages are still scaffold defaults and are not authoritative Reaktor documentation.
